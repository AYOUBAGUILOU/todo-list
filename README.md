## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn package manager

### Installation

1. Clone the repository:
```bash
git clone https://github.com/AYOUBAGUILOU/todo-list.git
cd todo-list
```

2. Install dependencies:
```bash
npm install
```

3. Start the application:
```bash
npm start
```

The application will open in your default browser at `http://localhost:3000`

## Usage

1. **Add a Task**: Type your task in the input field and press Enter or click the Add button
2. **Complete a Task**: Click the checkbox next to a task to mark it as complete
3. **Edit a Task**: Double-click on a task to edit it
4. **Delete a Task**: Click the trash icon to remove a task
5. **Filter Tasks**: Use the filter buttons to view all, active, or completed tasks
